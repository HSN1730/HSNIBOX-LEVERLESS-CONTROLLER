# HSNIBOX-LEVERLESS-CONTROLLER

**一个开源的3D打印无线全按键格斗游戏控制器。**

---

##  项目特性

- 2.4无线+USB有线低延迟双模芯片

- 外壳基于256*256平台FDM打印机优化分件

- 包含亚克力面板，增加表面质感

- 支持基于樱桃MX轴的热插拔轴座

---

##  项目详情

本项目的设计初衷是为了打造一款兼顾**便携性**、**低延迟**和**易制作**的无线无摇杆控制器。
<br>
### 1.  关于无线方案

本项目没有自己造轮子（开发固件），而是选择“站在巨人的肩膀上”。

芯片采用B站up主【**IMAPCTX**】设计生产的2.4无线+有线双模集成电路板。2.4Ghz模式回报率为800hz；有线连接模式回报率为1000hz；5ms按键消抖；自适应跳频；自带充放电模块。

- **易用性**板子集成充放电模块，已刷写好固件。只需要用2.54排针连接到PCB，插电即可运行。

- **低延迟**该方案在 2.4Ghz 模式下实测平均延迟约为1.3ms。

- **低功耗**在使用2000mAh电池情况下理论续航可达到150+小时。超低待机功耗，确保随拿随用摆脱续航焦虑。

- **功能全**支持四种SOCD模式;三种方向输入切换。
<br>
### 2.  关于外壳结构

- 外壳采用分体式打印，所有零件尺寸不超过拓竹主流256平台机器尺寸。外观整体性强，装配简单。

- 电池仓最大可以容纳 **60x36x115** 的4000mAh 3.7V电池，理论最高可实现 **300+**小时的续航。

- 装配已榫卯为主，螺丝为辅，面板亚克力使用双面胶固定。
<br>
### 3. 关于PCB

- PCB为mx标准轴热插拔设计，芯片和数据线均采用模块化连接，降低焊接难度。

- 这次用到的芯片无法通过软件形式修改键位，所以按键布局固定为14+1+5形式。

- 考虑到方便焊接问题，本版本取消【jocers】logo灯，改为单颗5050灯珠的指示灯。

## 组装指南

详细组装指南后续我会以视频的形式上传到视频网站**【bilibili】**,详情请关注B站账号**【末日料理师】**。

### 3D打印:

 >300mm以上打印平台可打印【3MF】件夹中的**300x300.3mf**文件。

>拓竹A1/P1/X1等256平台打印【3MF】件夹中的**256x256.3mf**文件。

>外壳和接收器部分建议使用**PETG CF**慢速打印。打印参数建议使用**3**层墙；**0.16mm**层高；**25%** 填充。

>非见光部分建议使用**PETG HF**高速打印。打印参数建议使用**3**层墙；**0.2mm**层高；**25%** 填充。

>考虑到不同玩家打印机参数、款式、材料等都大相径庭。所以在建模时没有加入公差，请各位在打印时根据自己机器和材料的情况设定合适的打印公差以方便后续拼装。

>因为打印尺寸较大，开放式机型打印时PEI板建议涂胶或更换粘性更强的底板以防止翘边造成打印失败。

### PCB焊接:
 
>Type-C接口使用**4Pin**排线或任意单独小电线连接PCB后用热熔胶固定到外壳上。

>集成电路板用**2.54mm**间隔**1x20**排针和**1x4**排针连接到PCB上。

>安装5050灯珠时注意GND位置方向对齐。

>购买电池时记得备注**【mx1.25公口】**

### 组装:

>接收器外壳组装前请先进性假组测试，如果盖上之后仍有松动，可在底塞内部填充其他材料以顶住接收器芯片避免松动。
